base font-size: 14px

Em – Relative to the font-size of the element (2em means 2 times the size of the current font)

size 2em

Ex – Relative to the x-height of the current font (rarely used)

size 2ex

Ch – The ch unit sets the font-size relative to the width of the character "0".

size 2ch

Rem – Relative to font-size of the root element

size 2rem

vw, vh – Relative to 1% of the width/height of the viewport

vmin – Relative to 1% of viewport smaller dimension

vmax – Relative to 1% of viewport larger dimension
